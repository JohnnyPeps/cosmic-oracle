# cosmic-oracle
Audio assets and web hosting repository for The Cosmic Oracle, an interactive Twine experience

The Cosmic Oracle: A Twine Divination Experience

This repository contains the source code and audio assets for an interactive tarot and astrology game built in Twine (Harlowe).

Inspired by the philosophy of paganism, the natural world, and my wife, the game engine cross-references a player's astrological sign with a randomized draw from a 78-card Tarot deck to generate over 900 unique readings.

Audio Hosting Note: The ambient forest and mystical sound effects included in this repository are royalty-free tracks sourced from Pixabay. They are hosted locally alongside the Twine index.html to ensure stable, uninterrupted web playback via GitHub Pages.
